## Agent Classification Criteria

Include files when:

- The file is a synthetic example used to test the vault pipeline.
- The file documents how category-level rules and templates work.

Exclude files when:

- The file contains real personal notes, research material, credentials, or local assets.
- Another user-created category is more specific.
